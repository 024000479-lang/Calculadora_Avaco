# Calculadora_Avaco
- Qué hicimos:
Se elaboro un proyecto, el cual consistía en programar una “calculadora” en PSeint, la cual debería ser capas de realizar operaciones básicas (suma, resta, multiplicación y división) aparte de poder sacar el área de figuras geométricas (circulo, cuadrado, triangulo y trapecio), calcular el diámetro y radio de un círculo, realizar estadística básica (Media, mediana y moda) y, por último, la calculadora debería ser capaz de hacer código Fibonacci.
 - Cómo lo hicimos:
Lo hicimos con códigos ya realizados como equipo, previamente en C++, y Java, solo tuvimos que cambiar el lenguaje de esos códigos a el que entiende PSeint, se hicieron correcciones de errores y modificaciones especiales para asegurar la comodidad del usuario al momento de realizar su compilación.
Aparte se realizaron comentarios, para ayudar a próximos usuarios y programadores a entender porque se realizo cada cosa y cuál es la función en el código.
- Para qué lo hicimos:
Lo realizamos para poder demostrar y poder dar a conocer nuestras habilidades de programación en equipo, aparte de ponernos a prueba a nosotros mismos y explorar nuestras capacidades.
 - Qué dificultades que tuvimos en el proceso:
Hubo muchas dificultades al momento de realizar el proyecto, el primero fue hacer errores para casos especiales, como, por ejemplo, avisar al usuario si el numero que eligió es Fibonacci o no o en caso de estadística básica para darle la opción al usuario de parar cuando tenga los números que quiere.
