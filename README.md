# Calculadora_Avaco
Este proyecto consiste en la elaboración de una calculadora con características adicionales (suma, resta, multiplicación, división, moda etc.) bastante básica, El proceso inicio con la pregunta inicial, ¿con que iniciamos? Para después decidir hacer uso de material previamente realizado con ya ciertas características, de ahí fue cuestión de agregar las funciones faltantes con ayuda de IA y tutoriales para saber cómo integrarlos de la mejor manera el proceso fue bastante tardado ya que primero se tuvo que volver a revisar el código reciclado para ver donde había más errores y poder “arreglarlo”, después solo fue cuestión de agregar lo requerido.
la calculadora fue desarrollada utilizando el pseudocodigo pseint lo que permitió que el desarrollo del código fuera mas rápido y fácil de entender y hacer.
 el código se estructura en cinco partes:
operaciones básicas: aquí se encuentra la suma, resta, multiplicación y división, la lógica de funcionamiento de este cuadro que el usuario elija la operación básica de su agrado, esta le solicitara dos número(dos variables) para realizar el cálculo en el caso de la división este verifica que  no se divida entre cero para evitar errores. 
cálculo de áreas: área del círculo, cuadrado, trapecio y triangulo, aquí se usan operaciones geométricas básicas por ejemplo, circulo área = pi * r2, cuadrado a = lado2, triangulo a = (base * altura) / 2.
diámetro y radio: conversiones entre radio y diámetro, si por ejemplo el usuario elije el radio, la formula es la sig (diámetro = radio * 2), si llega a ingresar el diámetro se usa la sig (radio = diámetro/ 2).
estadística: sus cálculos básicos que son la media, mediana y moda, aquí en usuario en la parte de la mediana el usuario ingresa sus números y después de finalizar el programa automáticamente los ordena y le da el de en medio,  la media(PROMEDIO) es mas fácil ya que solo suma todo y lo divide entre N cantidad de números, la moda identifica cual numero se repite mas entre todos los ingresados. 
Fibonacci: generación de los terminas de la sucesión, la sucesión funciona en verificar todos los numeros ingresados, dividiéndolos  y revisando si sobra o no, aquí se usan estructuras de control como según, mientras.
para que estos bloques se repitieran cuantas veces quisiera el usuario se hizo uso del ciclo repetir esto para como se dijo antes realizar múltiples ejecuciones del código, para poder seleccionar entre la gran cantidad de opciones que hay, se colocaron varios hacer, que permiten seleccionar una opción numerada
también se hizo uso de varios comentarios, para explicar la intencion de cada pedazo de código así como pone en claro el propósito de cada variable diferenciándolas entre si , esto ayuda mucho al lector guiándolo para que si es necesaerio pueda modificar el código o simplemente saber que hace por ejemplo el bloque 2.
1. Operaciones básicas: Permitir realizar operaciones básicas como la suma, resta, multiplicación y división, este pequeño bloque es útil para resolver operaciones basicas, mas que nada verificar resultados rápidamente.
2. Cálculo de áreas geométricas: Calcular el área de figuras (círculo, cuadrado, triángulo, trapecio) usando fórmulas, esta Apoya el aprendizaje de geometría, facilitando visualizar cómo se aplican las fórmulas basicas.
3. Diámetro y radio: Convertir entre radio y diámetro de un círculo, según lo que el usuario necesite, este le permite al usuario resolver problemas relacionados con figuras circulares.
4. Estadística: media, mediana y moda: Analizar un conjunto de datos numéricos para obtener medidas estadísticas básicas, Este módulo es útil para interpretar información, realizar análisis simples y comprender conceptos clave de estadística descriptiva.
5. Sucesión de Fibonacci: Verificar si un número pertenece a la sucesión de Fibonacci y generar términos posteriores, Introduce al usuario en conceptos de sucesiones matemáticas, lógica iterativa y patrones numéricos, fomentando el pensamiento algorítmico.
